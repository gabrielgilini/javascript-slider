Usage

The Slider constructor (although it can be called as a function) takes two parameters: fatherId and options; and returns an Object with references to two functions: Slider::start() and Slider::stop().

The first parameter, fatherId, should be the id attribute of the element which contains all the 'slides'.

The options parameter, is an Object that can have the following properties: 

{
    changeDelay: 5,         // Time, in seconds, between slide changes
    slideStep: 10,          // Number of pixels that will be slid per `slideDelay'
    slideDelay: 10,         // Time, in miliseconds, between animations
    slidesTagName: 'div'    // TagName of the slide elements
}
The values presented above are the default ones.

Note: when constructed, the slider starts automatically. The start and stop functions are for pausing and resuming purposes.
